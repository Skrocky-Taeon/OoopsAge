개인정보 처리방침

나이맞히기 앱의 개인정보 보호 정책

최종 수정일: 2025년 5월 31일

1. 개인정보 수집 및 이용

중요 안내

나이맞히기는 사용자의 개인정보를 존중하며, 최소한의 정보만을 수집합니다.

수집하는 정보

  • 업로드된 이미지: 나이 분석을 위해 임시적으로 처리됩니다
  • 분석 결과: 사용자가 요청한 분석 결과 데이터
  • 세션 정보: 앱 사용 중 임시 식별을 위한 세션 ID

이용 목적

  • 이미지 기반 나이 추정 서비스 제공
  • 분석 결과 표시 및 PDF 다운로드 서비스
  • 서비스 개선을 위한 통계 분석

2. 개인정보 보관 및 삭제

자동 삭제 정책

업로드된 이미지는 분석 완료 후 24시간 내에 자동으로 삭제됩니다.

  • 이미지 파일: 분석 후 24시간 내 자동 삭제
  • 분석 결과: 사용자 세션 종료 시 삭제
  • 세션 데이터: 브라우저 종료 시 자동 삭제

3. 개인정보 제3자 제공

제3자 제공 금지

나이맞히기는 사용자의 개인정보를 제3자에게 제공하지 않습니다.

다음의 경우를 제외하고는 개인정보를 외부에 제공하지 않습니다:

  • 사용자의 명시적 동의가 있는 경우
  • 법령에 의해 요구되는 경우
  • 서비스 제공을 위한 필수적인 업무 위탁의 경우

4. 이용자의 권리

사용자는 다음과 같은 권리를 가집니다:

  • 열람권: 본인의 개인정보 처리 현황을 확인할 권리
  • 정정·삭제권: 개인정보의 정정이나 삭제를 요구할 권리
  • 처리정지권: 개인정보 처리 중단을 요구할 권리

5. 개인정보 보호 조치

  • 암호화: 이미지 전송 시 HTTPS 암호화 적용
  • 접근 제한: 개인정보 처리 시스템에 대한 접근 권한 제한
  • 자동 삭제: 보관 기간 경과 시 자동 삭제 시스템 운영

6. 쿠키 사용

나이맞히기는 다음 목적으로 쿠키를 사용합니다:

  • 세션 유지 및 사용자 구분
  • 서비스 이용 편의성 향상

쿠키 거부: 브라우저 설정을 통해 쿠키를 거부할 수 있으나, 서비스 이용에 제한이 있을 수 있습니다.
